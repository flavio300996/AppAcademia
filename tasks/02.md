# Template de Prompt para Criar uma Rota de API

Crie a rota `PATCH /workoutPlans/{id(uuid)}/days/{id(uuid)}/sessions/:id(uuid)`

## Descrição

Ela atualiza uma sessão de treino específico.

## Requisitos Técnicos

- Crie arquivos de rotas separadas no caminho: @src/routes/

## Autenticação

- Rota protegida.
- O usuário precisa ser o dono do plano de treino recebido.
